Principais Formatações

Negrito ou destaque

Nessa frase temos um termo em negrito usando a tag <b>

Nesta outra frase temos um outro termo em negrito ou em destaque, como queiram, usando a tag <strong> (semantica).

Itálico / Ênfase

Já nesta frase temos um termo em itálico usando a tag <i> (nāo semantica).

Nesta outra frase temos um termo em itálico ou ênfase, usanto a tag <em> (semantica).

Texto Marcado

Podemos criar trambem um texto marcado com a tag <mark>.

Se usarmos apenas a tag <mark> o texto marcado aparecerá em amarelo

, mas se quisermos mudar a côr do marca texto é só usar Css. Por exemplo: vou usar um texto marcado em verde limāo, usando o parâmetro style background-color.

Se quisermos que a cor dos textos marcados sejam todos de uma mesma cor é só criar uma tag <style> no cabeçalho e crio um seletor dentro de style usando background-color

Texto Grande e Pequeno

Estamos criando um texto com letras grandes e com letras pequenas neste parágrafo. Usando as tags <big> e <small>

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas nāo considerado, usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase, e indicar que ele foi aciocionado depois, usando a tag <ins>.

Existe tambem o texto sublinhado com a tag <u>, que é igual a tag <ins>, porém u nāo é semantica

Texto Sobrescrito

Para inserir coisas do tipo x2+3, usando a tag <sup>.

Texto Subscrito

Para inserir coisas do tipo H2O, usando a tag <sub>.
